Tags

Functions

  • 표·CSV·엑셀 등 구조화된 데이터를 해석해 적절한 차트 유형을 자동으로 선택
  • 축, 범례, 값 라벨, 제목 등 시각 요소를 시각화 가이드라인에 맞게 자동 배치하고 렌더링 파라미터를 생성

Target users

  • 데이터 분석가와 BI 담당자
  • 경영 기획·마케팅 등 차트 기반 보고서를 자주 만드는 실무자

Ideal use cases

  • 경영 보고서·IR 자료용 차트를 표 데이터로부터 빠르게 초안으로 만들 때
  • 색각 이상 친화적이고 프레젠테이션용으로 깔끔한 차트를 자동으로 구성할 때

Limitations

  • 통계적 해석·인사이트 도출·정책 제안 등은 수행하지 않고, 데이터 시각화와 차트 렌더링에만 초점을 둔다
