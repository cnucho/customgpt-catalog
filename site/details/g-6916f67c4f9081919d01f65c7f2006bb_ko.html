Functions

  • 비정형 아이디어를 구조화된 연구 설계로 변환
  • 연구 유형(학술, 설문, 프로그램 평가, 데스크 리서치)에 맞는 설계 자동 생성
  • 개념 모델, 가설, 조사 설계, 로직모델 초안 작성
  • Mermaid/GoJS 기반 개념도 시각화
  • 다음 단계(GPT 워크플로) 제안

Target users

  • 대학 연구자 및 대학원생
  • 조사 및 평가 담당자
  • 리서치 컨설턴트
  • R&D 정책기획자

Ideal use cases

  • 논문 아이디어를 구조화된 연구 설계로 발전시킬 때
  • 프로그램 평가나 설문조사 설계를 자동화할 때
  • 데스크 리서치의 기본 구조(목표, 범위, 질문, 자료원)를 짤 때

Limitations

  • 통계분석이나 결과 해석은 수행하지 않음
  • 경험적 데이터에 기반한 모델 수정은 불가(ResRefiner 등 후속 GPT 필요)
  • 문헌검색이나 종합분석은 별도 GPT(LitSearch, SourceFinder 등) 필요

Example commands

  • ‘디지털 리터러시가 대학생의 학업 성취에 미치는 영향’을 연구 설계로 바꿔줘.
  • 청년정책 프로그램의 평가 설계를 설계형식(YAML)으로 작성해줘.
  • 기후기술 시장 트렌드 데스크 리서치 프로젝트를 설계해줘.

Additional features

  • 다국어 지원(영/한 등)
  • 구조화된 YAML 파이프라인 출력
  • 자동 프로젝트 유형 감지 및 맞춤 설계
  • 모듈 기반 설계 엔진 (academic_research, survey_project, program_evaluation, desktop_research)
