SurveyPersona Meaning Engine – 퍼소나 의미 해석 엔진

여러 퍼소나가 설문 문항을 어떻게 이해하는지 의미·감정·인지 관점에서 분석하는 엔진.

URL

https://chatgpt.com/g/g-693766b243248191b496489f57d9e13b-surveypersona-meaning-engine

Tags

  • survey
  • questionnaire
  • cognitive-testing
  • persona

Functions

  • 대상 집단을 대표하는 다양한 퍼소나를 자동 생성하고 설문 문항을 각 퍼소나 관점에서 해석한다.
  • 문항 이해 과정에서 드러나는 감정, 숨은 가정, 인지적 부담과 오해 가능성을 찾아낸다.

Target users

  • 설문 문항 작성자
  • 조사 방법론 연구자

Ideal use cases

  • 새 설문 문항을 여러 유형의 응답자 퍼소나로 사전 점검할 때.
  • 어떤 표현이 애매하거나 인지적 부담을 주는지 찾고 싶을 때.

Limitations

  • 실제 응답자가 아닌 시뮬레이션 퍼소나에 기반하므로 현장 인지면접을 완전히 대체할 수 없다.

Example commands

  • 다음 문항을 농촌 고령층, 도시 20대 직장인, 자영업자 퍼소나로 각각 어떻게 해석하는지 보여줘: '가계 소득 수준에 얼마나 만족하십니까?'
  • 환경세 관련 문항을 여러 퍼소나가 어떻게 이해하는지와 오해 지점을 분석해줘.
  • 디지털 금융서비스 이용 빈도 문항에서 고령층이 겪을 수 있는 인지적 어려움을 정리해줘.
  • 전기차 구매 의향 문항의 숨은 가정과 취약 집단(저소득, 농촌)의 해석 위험을 찾아줘.
  • 정치적 지지 성향이 다른 퍼소나들이 동일한 지지율 문항을 어떻게 다르게 받아들이는지 비교해줘.

Additional features

  • 맥락 정보를 주면 이에 맞는 퍼소나와 감정 상태를 자동으로 설정한다.
  • 한 번에 여러 퍼소나의 해석 결과를 병렬로 출력한다.
