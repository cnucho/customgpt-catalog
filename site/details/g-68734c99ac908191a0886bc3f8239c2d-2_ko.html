가상데이터 설계기 · (자동번역) 가상데이터 설계기

현실 기반 다차원 가상데이터를 생성하고, 메타정보 포함 교차표를 ChartGPT(표해)용 JSON으로 출력합니다.

Functions

  • 주제 입력을 해석해 사회지표/변수를 자동 매핑
  • 현실성 있는 시뮬레이션 수치 생성(평균/표준편차 규칙 기반)
  • 조건변수 2개 이상 포함한 다차원 교차표 구조 설계
  • 단위·시점·출처 등 메타정보를 포함해 해석 가능한 표로 정리
  • 표해(ChartGPT) 연동 가능한 table_state + data 형태(JSON)로 출력

Target users

  • 데이터 시각화/리서치 실무자
  • 정책/사회지표 분석가 및 기획자
  • 대시보드/리포트 제작자
  • 교육용 데이터셋이 필요한 강사/학생

Ideal use cases

  • 지역·연령·성별 등 조건 변수를 포함한 사회지표 가상데이터 생성
  • 분석/시각화용 교차표를 빠르게 설계하고 JSON으로 내보내기
  • 실데이터 부재 상황에서 현실성 있는 시뮬레이션 데이터로 프로토타이핑
  • 비교/경향/구성/분포/관계 등 목적에 맞는 차트 유형 추천과 데이터 준비

Limitations

  • 생성 데이터는 시뮬레이션이며 실제 통계와 다를 수 있으므로, 의사결정에는 원자료 검증이 필요합니다.
  • 민감한 주제(의료/법률/금융 등)에서는 최신·공식 자료로 교차검증을 권장합니다.

Example commands

  • 서울/부산/대구의 2020~2025 행복도·삶의만족도 가상데이터를 연령대×성별로 교차표(JSON)로 만들어줘.
  • 재택근무 생산성을 직무×근무형태로 비교하는 grouped bar용 표를 만들어줘. 단위/출처/시점 메타도 포함해줘.
  • 표해에서 바로 그릴 수 있게 table_state와 data로만 출력해줘.

Additional features

  • 차트 유형 가이드(비교/경향/구성/분포/관계 등) 기반 추천
  • 교차표 표준 템플릿(병합/정렬/스타일/메타 포함) 준수
  • 시각적 일관성을 위한 컬러 팔레트 규칙 적용
