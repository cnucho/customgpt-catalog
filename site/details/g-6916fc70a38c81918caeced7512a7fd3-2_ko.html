Source Analyzer · (자동번역) Source 분석기

사용자가 제공한 문서/웹페이지에서 핵심 주장·수치·KQ 관련성을 추출해 YAML의 source_extracts로 구조화한다.

Functions

  • 사용자 제공 문서(본문/HTML)를 읽고 핵심 주장(key_points) 추출
  • 수치·통계·구체 사실(data_points)을 원문 기준으로 정확히 발췌
  • 각 key/data가 어떤 KQ와 연결되는지 relevance_to_KQs로 매핑
  • 결과를 YAML의 source_extracts에만 작성/갱신(단일 YAML 출력)
  • Structured Mode(structured_pipeline: true)에서 형식 엄수 및 최소 설명

Target users

  • 데스크 리서치/문헌조사 담당자
  • 근거 기반 보고서 작성자(정책/시장/학술)
  • 리서치 파이프라인에서 출처 추출 단계를 자동화하려는 팀

Ideal use cases

  • 여러 출처 문서에서 핵심 주장·수치만 빠르게 추출해 구조화할 때
  • KQ(Research Questions) 기준으로 출처 내용을 태깅/매핑해야 할 때
  • Synthesis 단계 전에 source_extracts를 표준 형식으로 정리할 때

Limitations

  • 새로운 출처를 찾거나 sources 목록을 수정/추가하지 않음
  • synthesis(종합/비교/결론) 생성 금지 — 추출만 수행
  • PDF/이미지/다운로드 파일은 직접 읽지 못하며 텍스트 업로드가 필요함
  • 출력은 ‘업데이트된 YAML 한 덩어리’로 제한(설명 최소화)

Additional features

  • KQ별 관련성(relevance_to_KQs) 구조화로 추후 Synthesis GPT 연계에 최적화
