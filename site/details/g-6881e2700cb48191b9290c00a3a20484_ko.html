통계찾기

(자동번역) English: 통계찾기

공식 출처 기반으로 통계자료·법령정보를 단계적으로 검색하고 신뢰도(A/B)와 출처를 함께 제시하는 검색 보조 GPT.

Functions

  • 요청 의도를 분석해 통계(기본) 또는 법령 모드로 라우팅(자동/확인)하여 검색
  • 공식 통계(KOSIS/통계청/정부부처/OECD/World Bank 등) 중심으로 자료 후보를 수집·정리
  • 법령(국가법령정보센터 등)에서 관련 조문·근거를 찾아 메타정보와 함께 제시
  • 단계별 응답(개괄 → 후보 → 상세)으로 탐색 지원 및 원문 확인 경로 안내
  • 환각 방지: 출처 미확인 정보 배제, 날짜·근거 명시

Target users

  • 정책·행정·연구 보고서 작성자
  • 법령 근거 확인이 필요한 실무자

Ideal use cases

  • 공식 통계 지표 탐색 및 출처 정리
  • 관련 법령 조항 및 확인 경로 확보

Limitations

  • 공식 원문 확인이 어려운 경우 확정적 단정 불가

Example commands

  • 한국 청년실업률 공식 통계 출처를 찾아줘.
  • 안전관리자 선임의무 관련 법령 조항을 알려줘.

Additional features

  • 통계/법령 모듈 자동 분기
  • 신뢰도 등급(A/B) 표시
