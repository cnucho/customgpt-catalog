TableBuilder Pro

통계·텍스트 입력을 표 전용 DSL로 구조화하고, 보고서용 표를 설계·정규화·렌더링해 주는 전문 TableBuilder GPT.

URL

https://chatgpt.com/g/g-693b7b5196648191bb672f07b23d1259-tablebuilder-pro

Tags

  • policy
  • statistics
  • research-assistant

Functions

  • 정규화된 table_input·text_table_input 스키마를 기반으로 통계표·텍스트표 구조를 정의한다.
  • 표 목적에 맞게 행·열·셀 구성과 정렬·강조 규칙을 설계하고 DSL 템플릿으로 고정한다.
  • 여러 개의 유사한 표를 batch_production_mode로 일괄 생성할 수 있도록 표 템플릿을 재사용한다.
  • 마크다운·엑셀 등 최종 산출물 렌더링 규칙을 제공하고, 필요한 경우 파이썬 코드 스니펫을 생성한다.
  • SPSS·엑셀·Python 출력 결과를 표 전용 입력 형식으로 정규화하는 절차를 안내한다.

Target users

  • 정책·연구 보고서에서 복잡한 표를 반복 생산해야 하는 분석가·연구자
  • 조사·통계 결과를 표 형식으로 일관되게 정리해야 하는 데이터·리서치 팀

Ideal use cases

  • 설문·통계 분석 결과를 table_input 스키마에 맞춘 표 구조와 DSL 템플릿으로 정의할 때
  • 장단점·특성 비교 등 텍스트 기반 내용을 체계적인 비교표·요약표로 설계할 때
  • 연령·지역·소득 등 동일한 축을 갖는 분포표를 여러 개 batch로 자동 생성할 때
  • SPSS·엑셀 출력 결과를 보고서용 마크다운/엑셀 표로 일관성 있게 변환할 때

Limitations

  • 원자료(raw data)를 직접 분석하거나 통계치를 계산하지 않고, 이미 계산된 결과·구조화된 입력만을 표 형식으로 정리한다.
  • 내용을 임의로 추정·창작하지 않고, 사용자가 제공한 입력 내 범위에서만 표를 설계·정규화한다.
  • 민감한 개인정보가 포함된 원자료는 직접 업로드하지 않고, 이미 익명화·집계된 형태로만 다루는 것을 전제로 한다.

Additional features

  • tablebuilder_dsl_core.yaml, tablebuilder_io_data.yaml 등 보조 스펙을 기반으로 일관된 DSL·입력 스키마를 유지한다.
