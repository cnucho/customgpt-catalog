SPI GPT (survey-policy_insight)

여론·설문조사와 언론담론을 통합 분석해 Evidence-based Policy Insight를 제공하는 정책분석 GPT.

URL

https://chatgpt.com/g/g-692926ff1bb881919f6db364e703e390-spi-gpt-survey-policy-insight

Tags

  • policy
  • statistics
  • research-assistant
  • survey
  • public-opinion

Functions

  • 조사방법론(Sampling·대표성·편향·측정오차) 진단
  • 여러 조사·언론보도를 통합한 Evidence Summary 작성

Target users

  • 정부·지자체 정책담당 공무원
  • 국회 보좌관·입법조사관
  • 여론조사 기관·리서치 회사 연구원
  • 정책연구소·싱크탱크 연구자
  • 데이터 저널리스트 및 언론사 정책 담당 기자

Ideal use cases

  • 여론조사·설문조사 결과를 정책 관점에서 해석하고 싶을 때
  • 같은 이슈에 대한 여러 여론조사를 비교해 일관성과 편향을 점검하고 싶을 때

Limitations

  • 실시간 여론조사 데이터 수집 기능은 없으며, 사용자가 제공한 자료와 공개 정보에 기반해 분석함.

Example commands

  • “이 여론조사 결과를 정책결정자가 볼 수 있도록 ‘조사품질 진단 + 정책적 함의 + 정책옵션’ 구조로 정리해줘.”
  • “이 이슈에 대한 서로 다른 3개 여론조사 결과와 최근 언론 프레이밍을 비교해, 어떤 정책 대응 옵션이 가능한지 정리해줘.”

Additional features

  • Survey-to-Policy 보고서 템플릿에 맞춘 구조화된 분석 초안 자동 생성
