문빛 LayerFixer

(자동번역) English: 문빛 LayerFixer

사용자 문장을 4개 레이어(논리·구조·문체·형식)로 진단하고, 요청한 범위만 교정하는 다층 글 교정 GPT.

Functions

  • Layer 1: 원문 기반의 핵심 논리·사실·방법론 타당성 점검 및 오류 지적
  • Layer 2: 주장·문단 전개의 구조 재정렬과 연결어/반복 개선
  • Layer 3: 사용자 고유 문체를 유지한 문장 표현(가독성·명료성·어휘 정확성) 개선
  • Layer 4: 용어/형식/인용/표 구성의 일관성 정리(APA/MLA/Chicago 등 관례 반영)
  • 수정 전/후 비교와 주요 변경 사유 제시, 수정 후 평가(문체 유지·가독성·구조 개선 등) 제공

Target users

  • 논문/보고서/기획서 등 구조적 글쓰기를 하는 학생·연구자·직장인
  • 문체를 유지하면서 논리·구조·표현을 단계적으로 다듬고 싶은 작성자

Ideal use cases

  • 원고의 논리 점검(비약/모순/정의 불명확/근거 부족)과 수정 제안 받기
  • 문단 흐름을 재배치해 주장 구조를 선명하게 만들기(서론-근거-반론-결론 등)
  • 문체는 유지하되 문장 호흡·어휘 정확성·가독성을 개선하기
  • 용어/인용/표/번호 체계를 학술 관례에 맞춰 통일하기

Limitations

  • 원문에 없는 사실을 임의로 추가하지 않으며, 제공된 텍스트 범위 내에서만 평가·수정합니다.
  • 최신 사건/통계 검증이 필요한 경우, 사용자가 근거 자료(링크/인용/데이터)를 제공해야 정확도가 높습니다.

Example commands

  • Layer 1로만 검토해줘. 이 문단에서 논리적 비약/정의 문제/근거 부족을 지적하고 대체 표현을 제안해줘.
  • Layer 2로 재구성해줘. 주장 순서를 재배열해서 '문제-원인-해결-기대효과' 구조로 바꿔줘.
  • Layer 3(단순 명료화)로 지금 문체 유지하면서 문장만 매끄럽게 다듬어줘. 전후 비교도 보여줘.
  • Layer 4로 표 제목/번호/용어/인용 형식을 APA 기준으로 통일해줘.

Additional features

  • 레이어별 개입 범위를 분리해 과도한 재서술을 방지
  • 사용자 문체를 기준 상태로 학습·보존(요청 없으면 문체 변형 최소화)
  • 수정 후 자체 평가(문체 유지율/가독성 변화/구조 개선 등) 요약 제공
