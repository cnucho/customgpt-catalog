DataPrepCodeGen

메타데이터 기반으로 공식통계용 SPSS 데이터 전처리 코드를 생성하는 GPT.

URL

https://chatgpt.com/g/g-69396281e0188191be94814202314ee2-dataprepcodegen

Tags

  • policy
  • statistics
  • research-assistant

Functions

  • 변수 라벨, 값라벨, 결측, 재코딩, 파생변수, 리쉐이프를 포함한 SPSS 구문을 완전 생성.
  • 메타데이터 불일치를 자동 감지하고 안전한 기본 가정을 적용해 주석으로 문서화.

Target users

  • SPSS로 공식통계 조사자료를 전처리하는 분석 담당자
  • 재현 가능한 데이터 준비 스크립트가 필요한 연구자

Ideal use cases

  • 코드북 수준의 메타데이터로부터 전체 SPSS 데이터 준비 스크립트 생성.
  • 라벨, 결측, 파생변수, 리쉐이프를 표준 규칙으로 일괄 처리.

Limitations

  • 통계 분석이나 모델링 코드는 생성하지 않음.

Example commands

  • 이 메타데이터로 VARIABLE LABELS와 VALUE LABELS를 설정하는 SPSS 코드를 만들어줘.
  • 연령 변수를 연령대 그룹 변수로 재코딩하되 원변수는 유지해줘.
  • DO IF를 사용해 대상자 플래그와 스킵 플래그를 생성해줘.
  • wide 형태 변수를 long 형태로 리쉐이프하는 SPSS 구문을 만들어줘.
  • 문항 묶음으로 지수 변수를 만들고 검증용 명령도 추가해줘.

Additional features

  • 가정과 메타데이터 이슈를 코드 주석과 함께 자동 기록.
