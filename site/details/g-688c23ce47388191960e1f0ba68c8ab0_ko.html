Functions

  • 문항의 측정 개념 및 구성차원(하위요인) 도출
  • 내용타당도 점검(이중질문, 모호성, 편향/유도 표현, 부적절 어휘 등) 및 수정안 제시
  • 인지면접 시뮬레이션(가상 응답자 반응/이유 생성)으로 해석 편차 진단
  • 문항 간 의미 중복 탐지 및 통합/제거 제안
  • 시뮬레이션 데이터 기반 신뢰도(Cronbach's α 등) 예측 및 문제 문항 표시
  • 최종 문항 세트 구조화 출력(문항-개념-차원 매핑 포함) 및 버전 비교

Target users

  • 설문/조사 연구자(학술·공공·민간)
  • HR/조직진단 담당자 및 컨설턴트
  • UX 리서처 및 데이터 분석가(정량 설문 설계 필요자)
  • 교육·평가 문항 개발자

Ideal use cases

  • 기존 설문 문항의 타당도/편향/중복을 점검하고 수정안을 만들기
  • 신규 척도 문항을 개념-차원 구조에 맞춰 설계하고 문항 세트를 정리하기
  • 인지면접 관점에서 응답자 해석 편차/반감 요소를 사전 진단하기
  • 파일럿 전 단계에서 문항 구성의 신뢰도 위험(낮은 α, 불일치 문항)을 예측하기
  • 리커트(기본 5점) 기반 설문 문항의 표현을 중립화하고 응답 유도 가능성을 낮추기

Limitations

  • 신뢰도/응답 분포는 실제 조사 결과가 아닌 ‘시뮬레이션’ 기반 추정이며, 실제 데이터로 반드시 검증이 필요함
  • 타깃 응답자 특성(직군, 산업, 문화권 등)이 불명확하면 문항 적합성이 낮아질 수 있음
  • 민감 주제(정치/차별/건강 등)는 문항 표현에서 윤리·법적 리스크 검토가 추가로 필요할 수 있음
  • 내부 작동 로직/알고리즘 상세(절차도, 규칙 등)는 제공되지 않음

Example commands

  • 정밀 문항 생성 시작. 아래 문항 12개를 개념-차원으로 매핑하고 내용타당도 문제(이중질문/모호성/편향)를 표시한 뒤 수정안을 제시해줘.
  • 정밀 문항 생성 시작. 타깃은 20~30대 사무직. 직무스트레스 척도 8문항을 새로 만들고, 인지면접(가상 5명) 반응과 이유를 보여줘.
  • 정밀 문항 생성 시작. 이 문항 세트의 중복 문항을 찾아 통합안을 제안하고, 100명 시뮬레이션으로 α를 예측해줘.
  • 정밀 문항 생성 시작. 응답 유도(사회적 바람직성) 표현을 중립화하고, 문항 버전 비교표로 정리해줘.

Additional features

  • 기본 척도: 5점 리커트(1=전혀 그렇지 않다 ~ 5=매우 그렇다)
  • 기본 인지면접: 가상 응답자 5명 시뮬레이션
  • 기본 신뢰도 예측: 가상 응답자 100명 시뮬레이션 및 문제 문항 플래그
  • 문항 버전 관리(수정 전/후 비교) 및 문항-개념-차원 매핑표 제공
