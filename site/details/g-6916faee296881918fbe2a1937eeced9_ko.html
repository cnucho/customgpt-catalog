Source Finder

(자동번역) English: Source Finder

데스크 리서치를 위한 신뢰할 수 있는 출처를 웹에서 찾아 구조화된 목록(YAML)으로 정리합니다.

Functions

  • 키 질문/범위/목표에 맞는 신뢰 가능한 1차·2차 출처를 검색·선별
  • 각 출처를 메타데이터(제목, 링크, 유형, 연도, 간단 메모)로 구조화하여 sources 필드에만 채움

Target users

  • 정책·시장·기술 리서처
  • 보고서/논문 작성을 위한 참고문헌·근거자료가 필요한 실무자 및 학생

Ideal use cases

  • 리서치 질문에 대한 핵심 참고문헌/공식문서/통계/보고서 출처 목록 수집
  • YAML 프로토콜 기반 데스크리서치에서 sources 섹션만 빠르게 채우기

Limitations

  • 내용 요약·핵심 인사이트 도출·보고서 작성은 하지 않음(출처 목록만 제공)
  • objectives/scope/key_questions 및 source_extracts는 수정·작성하지 않음

Example commands

  • 아래 YAML의 objectives/scope/key_questions를 읽고, sources 필드만 S1, S2...로 채워줘.
  • 주제: 'AI 규제 동향'. 2020년 이후 정부/국제기구 1차 문서와 주요 연구보고서 12개 출처를 sources로만 정리해줘.

Additional features

  • 국문/영문 혼합 질의에서도 출처를 언어권별로 균형 있게 제시
