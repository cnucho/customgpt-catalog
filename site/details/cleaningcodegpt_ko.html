CleaningCodeGPT

(자동번역) English: CleaningCodeGPT

데이터 품질 규칙을 정규화하고 비파괴적 QC 검증 코드를 생성하는 GPT.

Tags

  • policy
  • statistics
  • research-assistant

Functions

  • 메타데이터를 canonical QC 규칙 JSON으로 정규화
  • R, SPSS, Python용 QC 검증 코드 생성

Target users

  • 데이터 관리자 및 통계 분석가
  • 조사·연구 데이터 품질 관리 팀

Ideal use cases

  • QC 메타데이터를 표준 규칙 정의로 변환
  • 원자료를 수정하지 않는 재현 가능한 QC 코드 생성

Limitations

  • 데이터 정제, 변환, 분석 기능은 제공하지 않음

Example commands

  • qcmeta와 lgmeta를 canonical QC rules JSON으로 변환해줘
  • 이 canonical_rules_json을 기반으로 R QC 코드를 생성해줘
  • SPSS용 QC FLAG 코드만 출력해줘
  • rules_only 모드로 QC 규칙 정규화 결과를 보여줘
  • 메타데이터 구조 오류가 있으면 error_report를 생성해줘

Additional features

  • 데이터 변경을 금지하는 엄격한 QC 코드 생성 정책
