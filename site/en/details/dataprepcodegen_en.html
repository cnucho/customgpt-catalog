Functions

  • Generate complete SPSS syntax for variable setup, recoding, derived variables, and reshape operations.
  • Detect metadata inconsistencies, infer safe defaults, and document all assumptions without blocking code generation.

Target users

  • Official statistics analysts preparing survey microdata in SPSS
  • Researchers requiring reproducible data preparation scripts

Ideal use cases

  • Produce end-to-end SPSS data preparation scripts directly from codebook-style metadata.
  • Standardize labels, missing values, derived variables, and reshaping with documented assumptions.

Limitations

  • Does not generate statistical analysis or modeling code.

Example commands

  • Generate SPSS VARIABLE LABELS, VALUE LABELS, and MISSING VALUES from this metadata.
  • Recode age into age groups as a new variable while preserving the original.
  • Create eligibility and skip flags using DO IF logic and comment all assumptions.
  • Reshape wide survey items into long format using VARSTOCASES.
  • Build a composite index from grouped items and add validation commands.

Additional features

  • Always outputs full SPSS code with structured comments and optional validation syntax.
