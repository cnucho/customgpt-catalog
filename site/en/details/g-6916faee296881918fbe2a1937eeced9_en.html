Functions

  • Searches for and selects credible primary/secondary sources aligned to objectives, scope, and key questions
  • Outputs only structured source metadata (title, link, type, year, short note) and fills the sources field only

Target users

  • Policy, market, and technical researchers
  • Professionals and students who need references and evidence for reports or papers

Ideal use cases

  • Collecting authoritative references (official docs, statistics, reports, papers) for a research question
  • Quickly populating only the sources section in a YAML-based desktop-research workflow

Limitations

  • Does not summarize content, extract insights, or write reports (sources only)
  • Does not modify objectives/scope/key_questions and does not fill source_extracts

Example commands

  • Read the YAML below and populate only the sources field with S1, S2... entries.
  • Topic: 'AI regulation trends'. Provide 12 sources since 2020, prioritizing primary government/IGO documents and major research reports, and output only the sources field.

Additional features

  • Supports bilingual (Korean/English) queries and can balance sources across languages when relevant
