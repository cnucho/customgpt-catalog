ScaleInsight · (자동번역) ScaleInsight

스케일·지수·KPI를 설계하고 문항 개발부터 타당도/신뢰도 검증 계획까지 지원하는 정밀 측정설계 GPT.

Functions

  • 개념 정의 및 차원(construct/dimension) 분해를 통한 측정모델 설계
  • 기존 문항/척도 맵핑 및 보완(채택·변형·차별화 논리 포함)
  • General/Behavioral 두 수준의 문항 자동 설계 및 수준 전환(추상화/구체화)
  • 스케일/인덱스/KPI 산출을 위한 득점 규칙을 수식 형태로 설계(z-score, 가중합 등)
  • 타당도(내용·구성·예측) 및 불변성/DIF 포함 검증 로드맵 제시(CFA/IRT 등)
  • 신뢰도 분석 설계(α, ω, CR, AVE, test-retest) 및 수치 기반 해석 가이드 제공

Target users

  • 리서처(학술/정책/평가) 및 설문 도구 개발자
  • 데이터/전략/HR/CS/제품팀의 KPI·지표 설계 담당자
  • 교육·평가·진단 도구 개발 조직(학교, 기관, 기업)

Ideal use cases

  • 신규 척도 개발: 개념정의 → 차원화 → 문항풀 → 파일럿 검증 계획까지 원스톱 설계
  • 기존 척도 개선: 문항 품질 점검, 역문항/편향 이슈 정리, 단축형(short-form) 설계
  • 인덱스 구축: 다차원 지표를 표준화/가중합해 0–100 지수로 운영(민감도 분석 포함)
  • KPI 설계: 측정지표+목표치(target)+해석 로직(등급/경보)을 포함한 운영안 수립
  • 타당도 검증 계획서 작성: 기준치(CFI, RMSEA 등)와 분석 단계/샘플 설계 정리

Limitations

  • 실제 α, ω, CFA 적합도 등 수치 산출은 응답데이터(파일럿/본조사)가 있어야 확정 가능
  • 민감한 개인/조직 데이터는 비식별화·요약 통계 형태로 제공하는 것이 권장됨
  • 문헌·기존 척도 비교는 제공된 자료/범위에 의존하며, 필요 시 출처 확인이 요구될 수 있음

Example commands

  • ‘AI 의존’ 개념을 3~4차원으로 정의하고, General/Behavioral 문항 각 5개씩 설계해줘. 7점 리커트로.
  • 고객경험(CX) 지수를 0–100으로 만들고, 득점수식(가중치 포함)과 민감도 분석 방법을 제시해줘.
  • 기존 20문항 척도를 10문항 단축형으로 줄이기 위한 기준(IRT/CFA)과 절차를 설계해줘.
  • 직무몰입 KPI를 측정지표/목표치/등급기준까지 포함해서 운영안으로 만들어줘.

Additional features

  • 문항-차원-척도-버전(General/Behavioral) 표 형태 결과물 제공
  • 기존 모델/지수와의 비교표(채택·변형·차별성) 포함
  • 득점 규칙을 반드시 수식으로 제시하고, 해석 기준(예: 상/중/하 컷오프)까지 제안
  • 검증 기준치 템플릿 제공(예: CFI>.90, RMSEA<.08, α>.70 등)
