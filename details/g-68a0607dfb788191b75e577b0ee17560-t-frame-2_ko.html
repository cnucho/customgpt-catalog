모형판별기 T-Frame · (자동번역) 모형판별기 T Frame

조사 목적에 맞는 이론적 프레임워크(행동·채택·마케팅 등)를 추천·비교·검토해 최적 모형을 함께 고르는 연구 보조 GPT.

Functions

  • 조사/연구 목적을 구조화하고 목적-객체(objective_object)로 정리
  • 목적에 맞는 이론/모형(예: TPB, HBM, UTAUT, COM-B 등) 후보를 추천하고 강점·한계·적합성을 비교
  • 가상 전문가 토론(상이한 관점)과 소비자 페르소나 시뮬레이션으로 적용 가능성 검증
  • 최종 선택 모형의 구성요소·트리·Mermaid 다이어그램·근거 문헌을 포함한 보고서 형태로 출력

Target users

  • 리서처/대학원생
  • 정책·캠페인 기획자
  • 마케팅/UX/프로덕트 전략 담당자

Ideal use cases

  • 행동 변화/기술 채택/구매 여정 등 주제에 맞는 이론 모형을 빠르게 탐색·비교하고 설계 근거를 마련
  • 설문/인터뷰/실험 설계 전 단계에서 변인(구성요소) 후보를 정리하고 개념적 모형을 설계
  • 보고서/제안서에 들어갈 ‘모형 선택 사유’와 문헌 근거를 체계적으로 정리

Limitations

  • 통계적 추정·인과 식별 자체를 수행하는 통계모형이 아니라, 개념적/이론적 프레임워크 추천에 초점
  • 사용자 맥락(대상, 행위, 채널, 제약조건) 정보가 부족하면 추천의 정밀도가 낮아질 수 있음
  • 문헌 링크는 가능한 한 1차/권위 출처 중심으로 제시하나, 접근권한(유료/기관 구독)에 따라 원문 열람이 제한될 수 있음

Example commands

  • ‘대학생의 친환경 소비 행동’을 설명할 모형 후보 5개를 추천하고 각각 강점/한계/적합성, 주요 구성요소, 대표 논문 링크를 정리해줘.
  • ‘사내 협업툴 도입’의 채택 요인을 연구하려는데 UTAUT와 TPB를 비교하고, 내 상황에서 어떤 변인을 우선 측정할지 제안해줘.
  • 최종 모형을 COM-B로 잡았을 때, 구성요소 트리와 Mermaid 다이어그램을 포함한 1페이지 요약 보고서를 만들어줘.

Additional features

  • 내장 모형 카탈로그(v2) 기반으로 TPB/HBM/UTAUT/AIDA/COM-B 등 다수 프레임워크를 빠르게 참조
  • 대화 흐름: 목적 수집 → 후보 추천 → 전문가 토론 → 소비자 시뮬레이션 → 최종 보고서
