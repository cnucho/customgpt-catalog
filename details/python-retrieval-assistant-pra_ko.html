Python Retrieval Assistant (PRA)

특정 문서·표·데이터셋·API 자원을 안정적으로 가져오는 저오류 파이썬/Colab 스크립트를 생성한다.

Functions

  • 알려진 URL에서 PDF/CSV/ZIP 등 파일을 내려받는 Colab/로컬용 파이썬 스크립트를 에러 처리 포함으로 생성.
  • 웹페이지 HTML 표를 CSV로 추출(BeautifulSoup/pandas)하고, 더 안정적이면 API 방식도 선택지로 제안.

Target users

  • 재현 가능한 데이터/문서 수집 스크립트가 필요한 연구자·분석가.
  • 공개 통계·보고서·표 데이터를 모으는 정책/학술/언론 조직.

Ideal use cases

  • 지정된 보고서/데이터 URL을 Colab 또는 로컬에서 다운로드하고 깔끔한 파일명·로그·재시도 옵션으로 저장.
  • 공식 사이트의 표를 CSV로 스크래핑하고(가능 시) 더 안정적인 API 대안까지 함께 제시.

Limitations

  • 광범위한 탐색형 리서치는 하지 않으며, 구체적인 대상(URL/페이지/API)이 필요하고 인증/유료장벽을 우회하지 않는다.
