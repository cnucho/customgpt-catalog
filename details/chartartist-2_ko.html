ChartArtist · (자동번역) ChartArtist

데이터 구조에 맞춰 정확하고 시각적으로 완성도 높은 차트를 자동 렌더링하는 GPT.

Tags

Functions

  • 데이터 형태에 따른 차트 유형 자동 결정
  • 엄격한 디자인 검증을 통한 고품질 시각화 렌더링

Target users

  • 데이터 분석가 및 연구자
  • 기획자 및 컨설턴트

Ideal use cases

  • 원시 데이터로부터 보고서용 차트 생성
  • 프레젠테이션과 문서에 일관된 시각화 적용

Limitations

  • 통계적 해석이나 인사이트 설명은 제공하지 않음

Example commands

  • 이 CSV 데이터를 기준으로 값 라벨이 포함된 막대 차트를 만들어줘.
  • 이 시계열 데이터를 색각 이상자 안전 팔레트로 선 차트로 표현해줘.
  • 비율 데이터를 최대 8개 항목의 파이 차트로 생성해줘.
  • 이 두 수치 변수의 관계를 추세선이 포함된 산점도로 시각화해줘.
  • 기본 HTML 템플릿을 사용해 차트 보고서를 출력해줘.

Additional features

  • 명도 대비 및 접근성 기준을 포함한 자동 검증 기능
