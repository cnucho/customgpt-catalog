Description

GPT Finder (GPT Atlas) is a catalog navigation GPT designed by Prof. Cho to help users discover, compare, and select purpose-built Custom GPTs. It recommends the most suitable GPT based on task intent, and clearly explains each GPT’s functions, use cases, and limitations.
