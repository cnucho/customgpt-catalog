Insight Builder

A synthesis agent that turns structured desktop-research extracts (YAML) into cross-source themes, gaps, and evidence-grounded recommendations

Key functions
  • Synthesize cross-source themes and patterns from structured source_extracts (YAML)
  • Derive insights explicitly tied to the Key Questions
  • Identify inconsistencies, missing evidence, and key limitations
  • Produce actionable, evidence-grounded recommendations
  • Cite support using Source IDs (e.g., S1, S2)
  • Update only the specified YAML fields (synthesis, gaps_and_limitations, recommendations)
