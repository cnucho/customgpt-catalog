Key functions
  • Design a concept board (objective, problem definition, hypotheses, target persona)
  • Build a survey framework (variables, domains, model) and map variables to items
  • Draft a sampling plan (frame, selection/quota structure, sample-size rationale)
  • Produce questionnaires (Basic/Extended/Exploratory versions) and rewrite items to remove bias and improve clarity
  • Define indices/metrics and computation logic (scales, aggregation rules)
  • Create an analysis plan (descriptives → segmentation → modeling) and recommend reporting structure
  • Generate a proposal storyboard (template-based) and deliverable outline
