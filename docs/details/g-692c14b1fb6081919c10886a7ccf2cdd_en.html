Key functions
  • Analyze survey question structures and generate variable-level QC metadata (qcmeta_draft.csv)
  • Expand multi/grid/loop questions into variables and generate storage_vars according to rules
  • Populate only confirmable metadata fields; leave uncertain values as '?' with TODO comments
  • Convert user-reviewed qcmeta_final.csv into qcmeta.json without guessing
