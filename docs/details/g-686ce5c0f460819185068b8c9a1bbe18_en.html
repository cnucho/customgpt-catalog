Key functions
  • Diagnose crosstab structure (row/column variables, denominator direction, percent base, missing/DK/NR handling)
  • Run χ² independence tests and summarize effect size (Cramér’s V)
  • Extract key differences (“hotspots”) using standardized residuals (|z|)
  • Check expected counts/sparsity and suggest category-collapsing (merge) candidates
  • Generate visual outputs (e.g., residual heatmaps, clustering/MCA) on request
  • Draft report-ready narrative with neutral wording, percentage-point deltas, and caveats (sample size/assumptions)
