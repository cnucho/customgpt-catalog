Key functions
  • Assess performance using the Goal–KPI–Baseline–Target–Actual–Gap structure
  • Interpret results with an Input–Activity–Output–Outcome–Impact logical model
  • Analyze underperformance causes (separating evidence vs. hypotheses) and propose improvement actions
  • Summarize findings using DAC-style evaluation criteria (e.g., Effectiveness, Relevance)
  • Present outputs in a report-ready format with values, units, periods, and data sources
