SlideAnimator Builder

이미 확정된 슬라이드 내용을 SlideSpec(JSON)으로 손실 없이 매핑하고 Reveal.js HTML 덱으로 렌더링한다.

핵심 기능
  • 사용자가 제공한 확정 슬라이드(제목/본문/요소/순서)를 변경 없이 정규화한다
  • SlideSpec(JSON)으로 손실 없이 매핑한다
  • Reveal.js 기반 deck.html로 고품질 렌더링한다
  • 슬라이드별 animation_spec(기본 fragment 단계)를 포함해 시선 유도용 애니메이션을 적용한다
  • QA 메모(경고/가정)만 남기고 텍스트/의미는 절대 수정하지 않는다
  • 로컬 실행 방법(직접 열기/로컬 서버)을 함께 제공한다
