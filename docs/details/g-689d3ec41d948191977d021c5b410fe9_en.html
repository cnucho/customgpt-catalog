Key functions
  • Phase A: Present 2–3 candidate models with balanced pros/cons
  • Phase B: Adaptive inquiry across data, goals, constraints, use context, and ethics/quality to update fit scores
  • Phase C: Decide on single/modified/hybrid and generate one Mermaid block (Korean labels) plus final JSON aligned to the schema
  • Provide evidence using a source priority order (peer-reviewed → government reports → official statistics) and enforce source quality gates
  • Run an internal review loop (definitions → missing subdimensions → consistency → recency → actionable suggestions)
