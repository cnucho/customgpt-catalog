핵심 기능
  • 설문 문항 구조를 분석해 단일 변수 수준의 QC 메타데이터(qcmeta_draft.csv) 생성
  • multi/grid/loop 문항을 규칙에 따라 변수 확장 및 storage_vars 생성
  • 확정 가능한 값만 메타 필드(type, valid_range, missing_codes 등)에 입력하고 불확실한 값은 '?' + TODO 코멘트로 표시
  • 사용자가 수정한 qcmeta_final.csv를 qcmeta.json으로 변환(임의 보정/추정 없음)
