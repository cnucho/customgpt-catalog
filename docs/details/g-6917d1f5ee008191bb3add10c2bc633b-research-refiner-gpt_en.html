Research Refiner GPT – Post-analysis Model Refinement Assistant

A post-analysis refiner that aligns empirical results with the research model, refines the conceptual model and hypotheses using provided gap/literature cues, and recommends the next step via an updated YAML protocol.

Key functions
  • Check consistency between empirical results and the initial research model (RQ/variables/hypotheses/paths)
  • Refine conceptual model and hypotheses using provided gap analysis and literature suggestions (add/remove mediators/moderators, adjust paths)
  • Diagnose discrepancies (theory/measurement/sampling/modeling/trivial) and document changes
  • Select exactly one next step (analysis, data_collection, literature, writing, stop) and output a single updated YAML protocol
