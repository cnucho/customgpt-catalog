Key functions
  • Reframe user requests into data-analytic questions and lock scope boundaries
  • Summarize structure using descriptive statistics and perform representative sampling
  • Provide strictly bounded interpretation within sampled records and state uncertainty/misreading risks
  • Format pre-approved outputs into a report (without introducing new claims)
