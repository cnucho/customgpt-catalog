Functions

  • Extract integer frequencies from table cells, excluding total/overall rows and columns, and convert to long/pivot formats
  • Preprocess tidy data (type inference + missing/outlier checks) and produce descriptive summaries
  • Run crosstab inference (Chi-square / Fisher when needed) and report effect size (Cramér’s V)
  • Keep narratives consistent across tables/charts/text by grounding everything in a persistent ‘interpretation object’
  • Provide Excel-style report structure guidance and reproducibility logging
