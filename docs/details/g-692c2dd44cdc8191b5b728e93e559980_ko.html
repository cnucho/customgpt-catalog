핵심 기능
  • 분석 목적·방법·변수 정보를 입력받아 전처리 규칙 초안을 생성
  • 결측/이상값/범주통합/재코딩/변환/스케일링/방법특이 규칙을 표준 순서로 설계
  • 각 규칙에 조건→행동→파라미터→근거(조사방법론·통계학 기준)를 명시
  • 규칙을 JSON/CSV 사양으로 동시 산출(구조·규칙 수 일치)
  • 사용자 승인 기반(초안→승인→최종) 규칙셋 생성 흐름을 지원
