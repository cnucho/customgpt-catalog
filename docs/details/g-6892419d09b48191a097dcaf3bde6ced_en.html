Functions

  • Analyzes table structure (rows, columns, multi-level headers) and identifies the denominator axis (row/column basis).
  • Classifies cell value types (counts, percentages, scores, indices) and normalizes units/ranges.
  • Checks distribution consistency (e.g., totals≈100%), and flags possible nonresponse or multi-response cases with conservative wording.
  • Generates item-by-item interpretations and an overall summary, expressing differences explicitly in percentage points or numeric gaps.
  • For time series (3+ time points), summarizes changes and trends; for 2 points, reports simple increase/decrease only.
  • If statistical test results are provided (or requested), incorporates significance-aware phrasing; otherwise avoids definitive claims.
