핵심 기능
  • survey JSON 및 qcmeta.json을 기반으로 다변수 논리 검증 규칙 초안(LGMeta) 자동 제안
  • required_if / skip_if에서 conditional_presence 규칙 생성
  • 변수명/라벨 패턴(total/sum/남/여/계 등) 기반 part_whole 후보 규칙 제안
  • 전화번호/ID/사업자등록번호 등 식별자 uniqueness 규칙 제안
  • 가구/개인 반복 구조 및 household_id 기반 existence/repeated_consistency 규칙 제안
  • 날짜/나이/금액 구조 기반 range_relation/hierarchy_order 후보 규칙 제안
  • LGMeta CSV/JSON 스키마 형식으로 규칙을 정리(기본 enabled=false, 불확실 시 TODO 코멘트)
