SurveyPersona Expression Engine – 퍼소나 발화 생성 엔진

퍼소나의 의미·감정을 받아 실제 응답자처럼 자연스러운 말투로 여러 답변 문장을 생성하는 엔진.

핵심 기능
  • 퍼소나 의미(JSON 또는 요약 텍스트)를 받아 자연스러운 한국어 구어체 응답으로 변환한다.
  • 동일한 의미를 유지하면서 길이·뉘앙스가 다른 여러 표현(변형)을 생성한다.
